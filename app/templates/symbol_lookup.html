Status Meanings:
  • Valid - Symbol found with current price
  • Warning - Symbol found but price unavailable
  • Error - Symbol not found or API error
F&O Availability:
  • Yes: Options contracts are available for this symbol
  • No: Only equity trading available (no derivatives)
Common Issues:
  • Symbol not found: Check spelling and ensure it's listed on NSE
  • No F&O: Not all stocks have futures & options contracts
  • Price N/A: Market may be closed or symbol suspended
  • API errors: Temporary network/authentication issues
Tips:
  • Use exact NSE symbols (e.g., "RELIANCE" not "RIL")
  • Symbols are case-insensitive
  • Check during market hours for live prices
  • Only F&O symbols can be processed for options data
